One‑click HTTP security headers with beginner‑friendly presets and a 60‑second onboarding wizard.

== Description ==

**Security Headers Boss (SHB)** adds the most important browser headers (HSTS, CSP, Referrer‑Policy, COOP/COEP)
to your WordPress site. It packages them into three simple modes:

* **Standard (Safe & Compatible)**
* **Strict (Blocks inline scripts)**
* **Paranoid (Maximum isolation)**

No code knowledge needed. Activate → pick a mode → you're protected.

== Installation ==

1. Upload the `shb` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the “Plugins” menu in WordPress.
3. Visit *Settings → Security Headers* and run the wizard.
